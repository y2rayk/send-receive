# send-receive
Build a service for sending and retrieving messages.

to build:
```
python3 -m venv venv
. ./venv/bin/activate
pip install -r requirements.txt
```

to run unit tests:
```
. ./venv/bin/activate
python manage.py test
```

to run server:
```
. ./venv/bin/activate
python manage.py runserver 
```
